# 🌱 Learning Spring Framework

A comprehensive repository for learning the **Spring Framework** from beginner to advanced level. This repository contains practical examples, hands-on projects, and notes to understand the core concepts of Spring and build enterprise Java applications.

---

## 📚 Topics Covered

- ✅ Introduction to Spring Framework
- ✅ IoC (Inversion of Control)
- ✅ Dependency Injection (DI)
- ✅ Spring Beans
- ✅ Bean Scopes
- ✅ Bean Lifecycle
- ✅ Autowiring
- ✅ XML Configuration
- ✅ Java-based Configuration
- ✅ Annotation-based Configuration
- ✅ Spring Expression Language (SpEL)
- ✅ Spring MVC
- ✅ Spring JDBC
- ✅ Exception Handling
- ✅ Validation
- ✅ AOP (Aspect-Oriented Programming)
- ✅ Best Practices

## 🎯 Learning Goals

- Understand the fundamentals of the Spring Framework.
- Learn how Dependency Injection works.
- Build loosely coupled and maintainable applications.
- Explore different configuration styles (XML, Java Config, Annotations).
- Develop Spring MVC applications.
- Connect applications with databases using Spring JDBC.

## 🚀 Getting Started

### Prerequisites

- Java JDK 17 or higher
- Maven
- IntelliJ IDEA (or any Java IDE)

### Clone the Repository

```bash
git clone https://github.com/<your-username>/Learning-Spring-Framework.git
```

### Navigate to the Project

```bash
cd Learning-Spring-Framework
```

### Build the Project

```bash
mvn clean install
```

### Run the Application

Run the desired example class from your IDE or use Maven if applicable.
